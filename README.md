# Desktop-2048
MacOS desktop version of the classic game 2048. 
Created using Swift 5.1, Apple SpriteKit, and Apple UIKit.


<img  src="Desktop2048/gameBackground.png" border="0" width="400">

## Key Components
- 1. Welcome scene
- 2. Game scene
- 3. Grid class
- 4. Aseets
